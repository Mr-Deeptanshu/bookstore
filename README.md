# 📚 Bookstore REST API

A simple Spring Boot REST API for managing bookstore data using H2 database.

---

## 🚀 Features

- Add new books
- Fetch all books
- In-memory H2 database
- JSON-based communication

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

---

## 📂 Project Structure

```
bookstore
 ├── src
 │   ├── main
 │   │   ├── java/com/example/bookstore
 │   │   └── resources
 │   └── test
 ├── pom.xml
```


---

## ⚙️ Installation & Setup

1. Clone the repository:https://github.com/Mr-Deeptanshu/bookstore
2. Navigate to project:cd bookstore
3. Run the application:mvn spring-boot:run
4. App runs on: http://localhost:8080

---
## 🌐 API Endpoints

| Method | Endpoint     | Description       |
|--------|-------------|------------------|
| GET    | /test       | Test API         |
| GET    | /add-test   | Add test data    |
| GET    | /books      | Get all books    |
| POST   | /add        | Add new book     |
